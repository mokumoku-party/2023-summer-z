## HANABISHI

夏休み...夏祭りで花火を見たいですよね？  
ということで、花火を見れるアプリを作りました。

https://hanabishi.deno.dev

- ビジョン
  - 誰でも手軽に、いろいろな花火を作って観察することができます
- ターゲット
  - 花火大会に行けなかった人
  - 花火大会で花火を見て、作り方や種類が気になった若者(10~20代)
